3.) cd Downloads
4.) cd ~/Downloads
5.) wget http://dlib.net/files/dlib-19.17.tar.bz2
6.) tar jxvf dlib-19.17.tar.bz2
7.) ls #you should found the dlib folder
8.) cd dlib-19.17
9.) gedit dlib/cuda/cudnn_dlibapi.cpp

Now a box will pop up. Find 'forward_algo = forward best algo' in the search bar in command it out using //. Then save it.
